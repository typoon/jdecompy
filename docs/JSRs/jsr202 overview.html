JSR 202 Proposed Final Draft


This is the proposed final draft describing proposed changes to the Java class file format under JSR202, slated for inclusion in Java 6. There are two PDF files. One is the revised version of chapter 4 of the class file format, which contains the modified specification. The other is a diff file that shows the differences visa vis the analogous Java 5 specification.

The main difference is the introduction of a new verification scheme based on type checking rather than type inference. This scheme has inherent advantages in performance (both space (on the order of  90% savings) and time (approximately 2x)) over the previous approach. It is also simpler and more robust, and helps pave the way for future evolution of the platform.
